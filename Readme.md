<details>
<summary>🔧 <b>How to activate the snake animation</b></summary>

<br>

1. In this repo, create the file `.github/workflows/snake.yml`
2. Paste the following content:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 5
    steps:
      - name: Generate Snake
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark
      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

3. Go to **Actions** tab → click **"Generate Snake Animation"** → **"Run workflow"**
4. Wait ~30 seconds, the snake will appear automatically!

</details>
